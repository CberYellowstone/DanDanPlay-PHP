# dandanplay-libraryindex

基于PHP重构的 弹弹play远程访问首页（媒体库内容的展示以及视频播放）

用于服务器部署

用到了DanDanPlay API

弹弹play每次更新时会附带此项目中master分支最新版的html/js/css和图片文件。
